## Description
Gist: Create a page that will have 3 dropdowns to save from the selected values in the URL.

## Local development

### Dependencies
* Node v14.8.2 and higher
* NPM v8.12.2 and higher

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
